# 27.移除元素

[力扣链接](https://leetcode.cn/problems/remove-element/description/)

```cpp
class Solution {
public:
    int removeElement(std::vector<int>& nums, int val) {
        int slow = 0, fast = 0;
        for (;fast<nums.size();fast++){
            if (nums[fast]!= val){
                nums[slow++] = nums[fast];
            }
        }
        return slow;
    }
};
```
