# 141.环形链表

[力扣链接](https://leetcode.cn/problems/linked-list-cycle/description/)

```cpp
class Solution {
public:
    bool hasCycle(ListNode *head) {
      ListNode *slow = head, *fast = head;
      while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
      }
      return false;
    }
};
```
