# 383.赎金信

[力扣链接](https://leetcode.cn/problems/ransom-note/description/)

```cpp
class Solution {
public:
    bool canConstruct(string ransomNote, string magazine) {
        int hash[26] = {0};
        for (char i:magazine) hash[i-'a']++;
        for (char i:ransomNote) hash[i-'a']--;
        for (int i:hash) if(i<0) return false;
        return true;
    }
};
```
