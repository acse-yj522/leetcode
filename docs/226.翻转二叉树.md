# 226.翻转二叉树

[力扣链接](https://leetcode.cn/problems/invert-binary-tree/description/)

```cpp
class Solution {
public:
    void traversal(TreeNode* root){
      if (!root) return;
      TreeNode* tmp = root->right;
      root->right = root->left;
      root->left = tmp;
      traversal(root->left);
      traversal(root->right);
    }

    TreeNode* invertTree(TreeNode* root) {
      traversal(root);
      return root;
    }
};
```
